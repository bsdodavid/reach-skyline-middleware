## Description

Reach Skyline Middleware is a Node.js-based backend service that handles user form submissions and stores data in a JSON file. It acts as a middleware between the frontend React application and persistent storage.

## Features

- Handles POST requests from frontend
- Stores user data in JSON format
- Express.js middleware configuration
- REST API architecture

## Tech Stack

- Node.js
- Express.js
- TypeScript
- React (Frontend)
- Git & GitHub
